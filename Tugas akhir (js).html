    Apa sih itu Javascript dan Jquery ? Apa juga fungsinya ?

         Hello world! Dalam pengembangan web, seringkali ditemukan sebuah sisipan script / kode yang berawalan (script) dan berakhiran (/script) ini. Inilah yang disebut dengan JavaScript. Javascript adalah sebuah bahasa pemrograman web yang berfungsi agar suatu laman web menjadi lebih interaktif dan dinamis.

    Javascript ini merupakan bahasa pemrograman berjenis Client Side Programming Language, maksudnya script ini berjalan di sisi klien, dilakukan oleh klien (klien yang dimaksud adalah peramban web, seperti Chrome, Firefox, Safari, IE, dsb). Penggunaan javascript ini bersifat opsional, jadi tidak harus ada pada suatu halaman web. Tetapi hampir semua laman web disisipi kode javascript. Mengapa bisa demikian? Ini karena dari fungsinya.

     
    Fungsi JavaScript

    Fungsi pokok dari pada bahasa JavaScript ini sendiri adalah membuat website menjadi lebih dinamis. Namun pada penggunannya, fungsi dari javascript menjadi lebih beragam, seperti validasi form, memberikan efek dan animasi tertentu, hingga membuat games bisa menggunakan javascript.

     
    Penulisan JavaScript

    Di bawah ini adalah cara menulis kode javascript secara sangat sederhana. 

    	<script type="text/javascript">
 	      alert('Hai sayang');
  	</script>

    Script di atas berfungsi untuk menampilkan box peringatan (alert box) pada peramban web Anda. Anda dapat mencobanya sendiri dengan menulis kode diatas pada notepad, simpan, dan buka menggunakan peramban web kesayangan.

    Nah, sekarang kita masuk ke bagian jQuery. Apa lagi nih jQuery? jQuery adalah sebuah library / framework javascript yang dibuat sedemikian rupa agar penggunaan javascript menjadi lebih ringkas dan mudah dan melakukan pekerjaan yang lebih banyak, seperti motto-nya "write less, do more". Terus, fungsinya apa? Apa pengaruh ke javascript.

    Fungsi dari jQuery ini sendiri adalah untuk memanipulasi konten-konten pada suatu halaman web HTML, menangani event, animasi, efek, dan memproses form menggunakan Ajax. Ajax adalah proses penanganan form menggunakan javascript dengan waktu yang lebih singkat dan tanpa meload ulang laman.

     
    Memulai Menggunakan jQuery

    Untuk mulai menggunakan jQuery, Anda diwajibkan untuk mengunduh jQuery library jquery.min.js dari website resminya. Atau juga bisa langsung menautkan kode <script> (javascript) pada URL library jQuery.Download jQuery-3.1.1.min.js . Setelah Anda mengunduh jQuery.min.js, letakkan file tersebut di dalam folder JS. Buka halaman yang ingin Anda beri jQuery menggunakan teks editor dan sisipakn kode dibawah ini di antara kode <head>.......</head> 

    	<script type="text/javascript"  src="js/jquery-3.1.1.min.js"></script>

    Setelah menuliskan kode diatas, saatnya menulis kode jQuery-nya. Letakkan kode berikut tepat di bawah kode di atas

    <script type="text/javascript">
            $("document").ready(function() {
                    $("#contoh").click(function() {
                            alert('Hai sayang');
                    });
            });
    </script>

    dan tuliskan kode di bawah ini di antara tag <body></body> 

    	<button id="contoh">Klik Saya!</button>

    Setelah Anda simpan dan buka menggunakan peramban web maka akan muncul sebuah tombol yang bertuliskan "Klik saya!". Dan jika Anda klik akan muncul dialog peringatan yang bertuliskan "Hai sayang". Baiklah, itulah beberapa contoh sederhana penulisan javascript dan jQuery, semoga bisa bermanfaat bagi Anda semua. Salam Koding!
