    Mengenal apa itu Bahasa Pemograman Java

         Hello world! Siapa yang tidak mengenal java? Java merupakan bahasa pemrograman tingkat tinggi yang berorientasi objek serta tersusun dari bagian di sebut kelas. Java ini di buat oleh Sun Microsyestms dan sekarang diakuisi oleh Oracle.Contoh sintaks dari Java 

    	// Outputs "Hello, world!" and then exits
 	      public class HelloWorld {
  	        public static void main(String args[]) {

   		      }
  	 	  }

     
    Fungsi Java

    Java berfungsi sebagai sekumpulan teknologi yang dipergunakan untuk membuat dan menjalankan perangkat lunak pada komputer standalone maupun lingkungan jaringan serta menjalankan semua program berbasis java seperti javascript.

     
    Sejarah Java

    Java pertama dibuat pada tahun 1990 dan di rilis tahun 1995.Java dikembangkan oleh James Golsing bersama Sun Microsysetms menggunakan pengembangan dari sintaks C dan C++ namun dengan lebih sederhana.Java terlahir dari The Green Project yang berjalan pada awal 1991 hingga 1992.Project tersebut di kerjakan oleh Patrick Naughton,Mike Sheridan,James Golsing serta 9 orang lainnya.

    Project ini menghasilkan sebuah produk bernama *7 (Star seven) namun gagal dipasarkan.Namun beberapa tahun kemudian Green project bersama sebuah perusahan berhasil membuat produk baru yang menjadi cikal bakal java.Pada tahun 1993 hingga pertengahan 1994 Green project memiliki nama baru First person yang bertujuan mengenalkan "oak" pada publik, namun usaha mereka gagal dan First person dibubarkan pada akhir 1994.

    Pada tahun 1995 "oak" berganti nama menjadi Java di karenakan nama "oak" telah digunakan oleh bahasa pemrograman lain.Nama Java diambil dari sebuah pulau di Indonesia dan logonya berasal dari kopi tubruk.versi awal dari Java adalah 1.0 dan menyertakan banyak paket standart awal dan terus dikembangkan.

     
    Kelebihan dan Kekurangan Java

    Java memiliki beberapa kelebihan pada programnya di antara lain :

    • Multiplatform. salah satu fungsi java adalah dapat dijalankan di mana saja dengan kelebihan pemrogram cukup menulis program java dan diubah dari bahasa yang dimengerti manusia menjadi bahasa mesin/bytecode.
    • OOP (Object Oriented Programming-Program berorientasi objek)
    • Perpustakaan kelas yang lengkap.Java terkenal dengan kelebihan perpustakaannya yang lengkap dan memudahkan pengguna dalam menjalankan program java.
    • Bergaya C+.Memiliki syntaks seperti bahasa pemrograman C++ yang membuat banyak pemrogram C++ pindah ke Java.
    • Pengumpul sampah otomatis.Memilki fitur pengaturan penggunaan memori sehingga kita tidak perlu mengaturnya.
    Selain kelebihan Java juga memiliki beberapa kekurangan pada programnya di antara lain :
    • Kurangnya dukungan antar platform.Java memiliki kelebihan multiplatform namun ternyata ada beberapa platform yang tidak kompatibel pada platform lain.
    • Mudah di dekompilasi.Kode pada pemrogram java dapat dengan mudah diubah dari kode jadi kembali ke kode sumber.
    • Penggunaan memori yang banyak.Berbeda dengan C/C++ Java menggunakan memori lebih besar dari keduanya.
