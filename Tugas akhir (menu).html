    Apa sih itu Javascript dan Jquery ? Apa juga fungsinya ?

         Hello world! Dalam pengembangan web, seringkali ditemukan sebuah sisipan script / kode yang berawalan (script) dan berakhiran (/script) ini. Inilah yang disebut dengan JavaScript. Javascript adalah sebuah bahasa pemrograman web yang berfungsi agar suatu laman web menjadi lebih interaktif dan dinamis.

    Javascript ini merupakan bahasa pemrograman berjenis Client Side Programming Language, maksudnya script ini berjalan di sisi klien, dilakukan oleh klien (klien yang dimaksud adalah peramban web, seperti Chrome, Firefox, Safari, IE, dsb). Penggunaan javascript ini bersifat opsional, jadi tidak harus ada pada suatu halaman web. Tetapi hampir semua laman web disisipi kode javascript. Mengapa bisa demikian? Ini karena dari fungsinya.
